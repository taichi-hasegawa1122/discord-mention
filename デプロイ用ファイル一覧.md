# エックスサーバーへのアップロード用ファイル一覧

## 必須ファイル

### サーバー側（Node.jsが必要な場合）

```
server.js
package.json
.env（トークンを設定）
node_modules/（またはサーバー側でnpm install）
```

### クライアント側（静的ファイル）

```
public/
  ├── index.html
  ├── style.css
  ├── app.js
  └── test.html（オプション）
```

## アップロード時の注意事項

1. **`.env`ファイルは絶対にアップロードしない**
   - トークンが含まれているため、セキュリティ上危険です
   - サーバーの管理画面で環境変数として設定してください

2. **`.gitignore`に含まれているファイルはアップロードしない**
   - `.env`
   - `node_modules/`（サーバー側でインストールする場合）

3. **開発用ファイルはアップロード不要**
   - `check-connection.js`
   - `refresh-servers.js`
   - `start.command`
   - `README.md`（必要に応じて）

## アップロード後の設定

1. **環境変数の設定**
   - サーバーの管理画面で`DISCORD_TOKEN`を設定
   - 必要に応じて`PORT`や`ALLOWED_ORIGINS`を設定

2. **Node.jsプロセスの起動**
   - SSHでサーバーに接続
   - プロジェクトディレクトリで`node server.js`を実行
   - バックグラウンド実行の場合は`nohup`や`pm2`を使用

3. **フロントエンドの設定**
   - `public/app.js`の`API_BASE`を本番環境のURLに変更
